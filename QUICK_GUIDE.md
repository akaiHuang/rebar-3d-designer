# 快速使用指南：從 DXF 匯入到 3D 建模

## ✅ 您的需求已實現！

現在系統支援：
1. **匯入 DXF 平面圖** → 自動顯示 2D 平面圖
2. **點擊任何區塊** → 選中牆體或柱子
3. **手動調整高度** → 每個元素可獨立設定高度

## 🎯 使用步驟

### 1. 匯入您的 DXF 檔案
```
點擊「匯入 DXF」→ 選擇 03f-cbs.dxf → 確定匯入
```

### 2. 查看 2D 平面圖
- 匯入後會先顯示黑色的 2D 平面圖線條
- 同時會生成低矮的 3D 元素（初始高度 100mm）

### 3. 選擇要調整的區塊
- **點擊任何牆體或柱子**即可選中
- 選中的元素會發藍光
- 左側面板顯示「選中元素高度」

### 4. 調整高度
**方法一：調整單一元素**
- 點擊選中某個牆體
- 拖動「選中元素高度」滑桿
- 即時看到高度變化

**方法二：批量調整**
- 拖動「整體樓高」滑桿設定高度
- 點擊「套用高度到全部」按鈕
- 所有牆體和柱子統一變高

### 5. 控制顯示
- ✅ **顯示 2D 平面圖**：勾選/取消顯示黑色平面圖線條
- ✅ **顯示網格**：輔助定位

## 💡 操作技巧

### 視角控制
- **旋轉**：左鍵拖曳
- **平移**：右鍵拖曳
- **縮放**：滾輪

### 快速視圖切換
- **俯視圖**：從上往下看（適合調整平面配置）
- **正視圖**：從側面看（適合調整高度）
- **3D視圖**：立體檢視

## 🎨 工作流程範例

### 場景一：繪製一般樓層
```
1. 匯入平面圖
2. 設定「整體樓高」為 3000mm
3. 點擊「套用高度到全部」
4. 開始配置鋼筋
```

### 場景二：不同高度的牆體
```
1. 匯入平面圖
2. 點擊外牆 → 調整為 3000mm
3. 點擊隔間牆 → 調整為 2400mm
4. 點擊女兒牆 → 調整為 1200mm
5. 配置不同區域的鋼筋
```

### 場景三：複雜結構
```
1. 匯入平面圖
2. 逐一點擊每個區塊
3. 根據實際設計調整高度
4. 切換到俯視圖檢查平面
5. 切換到3D視圖檢查立體
```

## 🔧 左側面板說明

### 建築模型控制
- **整體樓高**：設定預設高度（100-6000mm）
- **選中元素高度**：調整選中牆體/柱子的高度
- **顯示 2D 平面圖**：切換平面圖線條的顯示
- **套用高度到全部**：一鍵設定所有元素高度

### 鋼筋配置
- 選擇鋼筋類型（直鋼筋、箍筋等）
- 設定規格、長度、間距
- 雙擊空白處或按 A 鍵添加

## 📊 狀態欄資訊

底部狀態欄會顯示：
- **建築資訊**：`牆: X | 柱: Y | 層高: Zmm`
- **滑鼠座標**：即時位置
- **當前視圖**：俯視/正視/3D

## ⚠️ 注意事項

1. **首次匯入**：
   - 所有元素初始高度為 100mm
   - 目的是讓您先看到平面圖
   - 然後再手動調整高度

2. **選擇元素**：
   - 點擊 3D 元素才能選中
   - 選中後會發藍光
   - 一次只能選中一個元素

3. **2D 平面圖**：
   - 黑色線條在 Y=10mm 高度
   - 可隨時切換顯示/隱藏
   - 不影響 3D 元素

## 🚀 立即開始

1. 開啟瀏覽器 → `index.html`
2. 點擊「匯入 DXF」
3. 選擇 `03f-cbs.dxf`
4. 開始調整您的建築模型！

## 📝 常見問題

**Q: 為什麼匯入後看不到東西？**
A: 初始高度很低（100mm），請縮放視圖或調整高度。

**Q: 如何選中特定的牆？**
A: 直接用滑鼠點擊 3D 視圖中的牆體即可。

**Q: 可以儲存調整後的高度嗎？**
A: 點擊「儲存專案」會保存所有設定。

**Q: 調整高度後可以復原嗎？**
A: 可以再次點擊元素並調整滑桿。

---

**提示**：建議先用「套用高度到全部」設定基本高度，再針對特殊區塊單獨調整！
